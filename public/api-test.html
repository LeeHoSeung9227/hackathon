🚀 새로 추가된 API 테스트

1. 이름으로 사용자 검색

GET /api/users/search?name={검색어}

2. 개인 랭킹 Top 30/100

GET /api/rankings/individual/top?scopeType={스코프}&limit={30|100}

3. 단과대 랭킹 Top 30/100

GET /api/rankings/college/top?scopeType={스코프}&limit={30|100}

4. 포인트 내역 타입별 조회 (개선된 기능)

GET /api/points/user/{userId}/type/{changeType}
