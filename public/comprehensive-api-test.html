👤 User API

GET /api/users - 모든 사용자 조회

GET /api/users/{id} - 특정 사용자 조회

GET /api/users/search - 이름으로 사용자 검색 (새로 추가)

🏆 Ranking API

GET /api/rankings/individual/top - 개인 랭킹 Top N 조회 (새로 추가)

GET /api/rankings/college/top - 단과대 랭킹 Top N 조회 (새로 추가)

💎 Point API

GET /api/points/user/{userId} - 사용자 포인트 내역 조회

GET /api/points/user/{userId}/type/{changeType} - 타입별 포인트 내역 (개선됨)

🤖 AI API

GET /api/ai/user/{userId} - 사용자별 AI 분석 결과

GET /api/ai/images/user/{userId} - 사용자 업로드 이미지 목록

🔐 Auth API

GET /api/auth/signup/all - 모든 가입 신청 조회

🏠 Main API
