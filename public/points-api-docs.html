사용자 포인트 내역 조회 (타입별)

GET
/api/points/user/{userId}/type/{changeType}
📋 설명

특정 사용자의 포인트 변경 내역을 타입별로 조회합니다. 포인트 획득, 사용, 차감 등의 이력을 확인할 수 있습니다.

🔗 URL 파라미터
파라미터 타입 필수 여부 설명
userId Long 필수 조회할 사용자의 고유 ID
changeType String 필수 포인트 변경 타입 (EARNED, USED, DEDUCTED, EXCHANGED 등)
📤 요청 헤더
헤더 값 필수 여부 설명
Content-Type application/json 필수 응답 데이터 형식
Authorization Bearer {access_token} 필수 사용자 인증 토큰

🔍 포인트 타입 설명
타입 설명 포인트 변화
EARNED 포인트 획득 + (양수)
USED 포인트 사용 - (음수)
DEDUCTED 포인트 차감 - (음수)
EXCHANGED 상품 교환 - (음수)
BONUS 보너스 포인트 + (양수)

📝 비고
  • 로그인 API는 토큰을 발급하며, 이후 모든 요청은 Authorization 헤더가 필요합니다.
  • changeType이 all인 경우 모든 타입의 포인트 내역을 반환합니다.
  • 응답 데이터는 생성일시(createdAt) 기준으로 내림차순 정렬됩니다.
  • 포인트 값은 양수(획득) 또는 음수(사용/차감)로 표시됩니다.
